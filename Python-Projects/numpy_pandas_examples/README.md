# NumPy & Pandas Examples

This project contains Python scripts demonstrating data manipulation using NumPy and pandas, along with JSON parsing examples.

## Features
- Array operations with NumPy
- DataFrame creation and manipulation
- Basic statistics and data analysis
- Exporting cleaned data to CSV/Excel
- Advanced dataset manipulation with Pandas
- JSON parsing, writing, and reading examples

## Files

### `numpy_pandas_examples.py`
- Demonstrates filtering, aggregation, and merging datasets
- Covers basic Pandas operations and NumPy integration

### `pandas_data_sets_manipulating.py`
- Practical examples of dataset manipulation:
  - Sorting and renaming columns
  - Handling missing values
  - Grouping and summarizing data
  - Joining and merging multiple datasets
- Focuses on real-world workflows for data analysis

### `json_examples.py`
- Basic JSON parsing from strings
- Accessing nested JSON data
- Looping through JSON lists (common API responses)
- Writing JSON to file
- Loading JSON from file

## Tech Stack
- Python
- pandas
- numpy
- json (built-in Python module)

## How to Run
```bash
pip install pandas numpy
